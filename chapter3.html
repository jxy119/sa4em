【理论思考】

核心理念：矢量数据的层次化组织

  1. 空间参考统一性：所有图层必须基于同一坐标系，确保空间位置准确叠加

  2. 图层逻辑结构：按”基础-专题”层次组织，基础图层（边界、道路）在下，专题图层（建筑、设施）在上

  3. 属性表设计原则：

    • 字段类型匹配数据性质（文本、整数、小数）

    • 预留扩展字段（如importance防汛重要性）

    • 设置字段约束保证数据质量

  4. 拓扑关系维护：建筑应在校区边界内，道路应连接畅通，避免几何错误

制图表达的艺术与科学

  • 色彩语义：暖色系（教学楼）表人文，冷色系（水系）表自然，红色表危险/重要

  • 符号等级：通过大小、形状区分要素重要性

  • 标注策略：避免压盖，按视图缩放动态显示

  • 地图要素：指南针、比例尺、相关信息、地图布局……

【QGIS实战】

基础数据说明

  1. 底图参考：优先采用高德卫星影像作为底图参考。

    • 理想标准：国家官方”天地图”为最权威底图，但因需申请API令牌，流程较为复杂。

    • 实用方案：高德卫星影像数据开放易用，可作为本次练习的替代方案。

    • 坐标说明：高德影像使用GCJ-02坐标系（俗称火星坐标系），此为基于WGS-84的加密坐标系。虽与国家标准CGCS2000存在细微差异，但在精度要求下不高情况下，两者可近似互换使用。

  2. 坐标设置：项目坐标系统一设置为 EPSG:4527（CGCS2000 / 3-degree Gauss-Kruger zone 39）。

    • 国家规范与金陵地区最适用的坐标系

    • 确保所有矢量图层在统一空间基准下精准叠加

一、新建工程与基础设置

启动QGIS并创建新项目

关键步骤：

  1. 打开QGIS软件

  2. 文件 → 新建项目（Ctrl+N）

  3. 立即保存：文件 → 保存项目（Ctrl+S）

  4. 命名：“金陵国子监防灾底图.qgz”

  5. 选择合适存储位置

新建工程

设置项目坐标系

关键步骤：

  1. 项目 → 属性 → CRS（坐标系）

  2. 在筛选器中输入”4527”

  3. 选择”EPSG:4527 - CGCS2000 / 3-degree Gauss-Kruger zone 39”

  4. 点击”应用”

  5. 点击”确定”保存设置

验证：查看QGIS右下角状态栏，确认显示”EPSG:4527”

二、配置捕捉设置（绘图前需要做）

开启捕捉工具栏

  1. 视图 → 工具栏 → 确保”捕捉工具栏”已勾选

  2. 界面会出现捕捉工具栏（磁铁图标相关工具）

详细捕捉配置（这里需要大家一边画图，一边探索一下）

  1. 点击磁铁图标”启用捕捉”

  2. 点击右侧下拉箭头 → 捕捉选项

  3. 设置参数：

    • 容差：12像素
    • 单位：像素
    • 拓扑编辑：启用

  4. 捕捉类型：

根据需要选择捕捉类型，可以先把顶点捕捉和交叉点捕捉打开。
设定捕捉

三、按顺序创建各图层

创建学府疆界（面图层）

创建步骤：

  1. 图层 → 创建图层 → 新建Shapefile图层

  2. 文件名：浏览选择存储位置，命名”boundary.shp”

  3. 几何类型：选择”多边形”

  4. 坐标系：确保选择”EPSG:4527”

  5. 添加字段：

    • 点击”添加字段”按钮

    • 名称：name，类型：文本数据，长度：50

    • 名称：type，类型：文本数据，长度：20

    • 名称：area_m2，类型：十进制数，长度：10，精度：3

  6. 点击”确定”创建图层

新建边界图层

绘制步骤：

  1. 在图层面板选中”boundary”图层

  2. 点击工具栏”切换编辑”（黄色铅笔图标）

  3. 点击”添加面要素”工具（绿色多边形图标）

  4. 在画布上开始绘制：

    • 沿校区边界逐点单击

    • 注意观察状态栏坐标变化

    • 使用捕捉功能确保图形闭合

  5. 绘制完成后右键结束

  6. 弹出属性对话框：

    • name：金陵国子监

    • type：主校区

    • area_m2：暂时留空（后续计算）

  7. 点击”确定”

  8. 点击”保存编辑”（磁盘图标）

  9. 点击”切换编辑”退出编辑模式

边界绘图

计算属性（面积）（其他计算长度等类似，大家自己探索）：

  1. 右键boundary图层 → 打开属性表

  2. 点击”切换编辑模式”（黄色铅笔）

  3. 右键area_m2字段 → 字段计算器

  4. 勾选”仅更新选中的要素”

  5. 表达式区域输入：$area

  6. 点击”确定”计算面积

  7. 保存编辑并退出编辑模式

计算面积

创建绿地系统（面图层）

操作步骤：

  1. 图层 → 创建图层 → 新建Shapefile图层

  3. 几何类型：多边形

  4. 坐标系：EPSG:4527

  5. 字段设置：

    • type（文本，20）：草坪/林地/运动场/广场

  6. 确定创建

四、设置图层样式（上节课讲过的）

在图层面板中拖动图层排序

  1. boundary（最底层）

  2. green_spaces

  3. buildings

  4. roads

  5. water_system

  6. gate（最顶层）

  7. 高德卫星图（如有，可放最底层或关闭）

设置边界图层样式

  1. 右键boundary图层 → 属性 → 符号化

  2. 选择”单一符号”

  3. 点击符号预览进入符号编辑器：

    • 填充色：完全透明（Alpha=0）

    • 边框色：深红色 (#8B0000)

    • 边框宽度：1.2毫米

  4. 确定应用

添加标注

  1. 属性 → 标注 → 单个标注

  2. 值选择：name字段

  3. 设置字体：黑体，12pt，深红色

  4. 确定应用

其它图层也类似

五、地图布局与输出

创建打印布局

  1. 项目 → 新建打印布局

  2. 命名：“金陵国子监防灾底图”

  3. 在布局中添加：

    • 图例（各图层符号说明）

    • 指北针（方向指示）

    • 标题文本：“南京大学仙林校区防灾专题图”

  4. 调整各元素位置和样式

地图布局

  2. 选择输出路径和文件名

  3. 设置分辨率：300dpi（高质量输出）

  4. 数据备份：定期备份整个项目文件夹
